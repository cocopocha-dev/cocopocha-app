# CHRONO FLUX - 未来型タイマーアプリ

サイバーパンク風の未来的デザインを採用したカウントダウンタイマーWebアプリです。

## 機能

- **カウントダウンタイマー** - 時間・分・秒を指定してタイマーを開始
- **プリセットボタン** - 1分 / 3分 / 5分 / 10分 / 15分 / 30分をワンタップで設定
- **プログレスリング** - 残り時間を円形プログレスバーで視覚的に表示
- **アラーム通知** - タイマー完了時にサウンド付きのオーバーレイ表示
- **キーボードショートカット** - Space: 開始/一時停止、Escape: リセット
- **モバイル対応** - レスポンシブデザインでスマートフォンでも快適に使用可能

## 使い方

`index.html` をブラウザで開くだけで使用できます。サーバー不要です。

```bash
open index.html
# または
python3 -m http.server 8000
```

## 技術仕様

- Vanilla JavaScript（フレームワーク不使用）
- CSS3 アニメーション / カスタムプロパティ
- Web Audio API（アラームサウンド）
- SVG プログレスリング
- レスポンシブデザイン（モバイル対応）
